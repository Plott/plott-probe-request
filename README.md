# plott-probe-request

Parses probe request data using tcpdump


### `plott.probeRequest`

Scans surrounding area for probe requests.


### Example

```js
plott.probeRequest();
--Do something
})
```


**Returns** `Function`, 

## Installation

Requires [nodejs](http://nodejs.org/).

```sh
$ npm install plott-probe-request
```
